Objectivo:
Pretende-se que crie uma nova aplicação react com as seguintes 
características: aplicação para gestão de um restaurante.

==============================================================================================================

Requisitos:
➢
Sistema de registo de users
➢
Página para gestor do restaurante: Deve haver uma página destinada ao gestor 
do restauranre onde este insere na api 3 campos: entradas, prato principal, 
sobremesas. A rota para acesso deve estar protegida ao role gestor.
➢
Consumidor: A aplicação deve carregar menu com os dados vindos da api e 
permitir ao user selecionar os items que que e criar uma encomenda que a 
cozinha irá visualizar. A rota para acesso deve estar protegida ao role consumidor.
➢
Cozinha: Na aplicação deve haver uma rota denominada à cozinha em que aparece 
uma lista dos nomes da pessoa que faz o pedido e os items que lhe correspondem. 
Na cozinha deve ser possível actualizar o estado dos pedidos para: em confecção 
ou entregue.
